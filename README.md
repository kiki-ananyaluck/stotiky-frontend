## Getting Started

First, run the development Frontend:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3001](http://localhost:3000) with your browser to see the result.

This is BackEnd [https://github.com/kiki-ananyaluck/spotiky-backend]
